The Investors API provides structured data on institutional and individual investors operating across the private markets. Each profile includes fund type, assets under management, stage and sector focus, portfolio composition, and a chronological feed of recent investment activity â€” giving you the context to map the competitive funding landscape around any company or deal.

## Get an Investor Profile

**`GET /api/v1/investors/{investorId}`**

Retrieve the full profile for a single investor by their Specter ID. This endpoint covers venture capital firms, private equity funds, corporate venture arms, and active angel investors.

### Path Parameters

<ParamField path="investorId" type="string" required>
  The unique Specter ID for the investor. You can obtain this from a transaction record, a company profile's investor list, or a search result.
</ParamField>

### Example Request

```bash
curl --request GET \
  --url "https://app.tryspecter.com/api/v1/investors/inv_03k8pq5r7s2u9t4v" \
  --header "X-API-Key: YOUR_API_KEY"
```

### Response Fields

<ResponseField name="id" type="string">
  Specter's unique identifier for this investor. Use this ID when querying transactions or signals filtered by investor.
</ResponseField>

<ResponseField name="name" type="string">
  The investor's full firm name or display name for individual angels.
</ResponseField>

<ResponseField name="type" type="string">
  Investor category. One of: `vc` (venture capital), `pe` (private equity), `angel` (individual angel investor), or `corporate_vc` (corporate venture arm).
</ResponseField>

<ResponseField name="aum_usd" type="integer">
  Assets under management in USD, based on the most recent fund disclosures and public filings. May be `null` for angel investors or undisclosed funds.
</ResponseField>

<ResponseField name="investment_stages" type="array">
  The funding stages this investor actively participates in. Values correspond to standard round types such as `seed`, `series_a`, `series_b`, `series_c`, and `growth_equity`.
</ResponseField>

<ResponseField name="portfolio_count" type="integer">
  Total number of portfolio companies tracked by Specter for this investor, including both active and exited positions.
</ResponseField>

<ResponseField name="recent_investments" type="array">
  The investor's most recent deal activity, sorted by date descending.
  <Expandable title="recent_investments item fields">
    <ResponseField name="company_id" type="string">
      Specter ID of the portfolio company. Use this to retrieve the full company profile.
    </ResponseField>
    <ResponseField name="company_name" type="string">
      Name of the company that received the investment.
    </ResponseField>
    <ResponseField name="round_type" type="string">
      Round type of the investment, e.g. `seed`, `series_a`, `series_b`.
    </ResponseField>
    <ResponseField name="date" type="string">
      ISO 8601 date the investment was announced or closed.
    </ResponseField>
    <ResponseField name="amount_usd" type="integer">
      Investment amount in USD for this specific deal. May be `null` if undisclosed.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="hq" type="object">
  Headquarters location of the investor.
  <Expandable title="hq fields">
    <ResponseField name="city" type="string">
      City where the firm is headquartered.
    </ResponseField>
    <ResponseField name="country" type="string">
      Country of the firm's primary office.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="website" type="string">
  The investor's primary website URL.
</ResponseField>

<ResponseField name="linkedin_url" type="string">
  LinkedIn company or profile page URL for the investor.
</ResponseField>
