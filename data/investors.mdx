Specter tracks **330,000+ investors** — venture capital firms, private equity funds, family offices, corporate venture arms, angel networks, and sovereign wealth funds. Each profile goes beyond directory data: fund details, portfolio companies, stage preferences, and historical investment patterns.

## API endpoints

Everything above is available through the Investors API. Full request and response detail for each endpoint lives in the [Investors API reference](/api/investors).

| Method | Endpoint | Returns |
|---|---|---|
| `GET` | `/investors` | Look up an investor by website or name |
| `GET` | `/investors/search` | Find an investor by name |
| `GET` | `/investors/{investorId}` | A full investor profile by ID |
| `GET` | `/investors/{investorId}/funding-rounds` | Funding rounds the investor has participated in |
| `POST` | `/investors/by-id/bulk` | Look up up to 50 investors in one request |
