Specter tracks **330,000+ investors** — venture capital firms, private equity funds, family offices, corporate venture arms, angel networks, and sovereign wealth funds. Each profile goes beyond directory data: fund details, portfolio, historical investment patterns, and interest signals that surface which investors are actively evaluating opportunities before deals are announced.

## Interest Signals: Know Before the Deal

Specter detects when investors are **actively in deal-sourcing mode** — before a term sheet is signed or an announcement is made — by tracking observable engagement between investors and the companies and people they're evaluating. Each investor profile carries the IDs of any active interest signals, which you can pull through the Signals API.

<Note>
  Interest signals are most actionable combined with company-level tracking. Set up a watchlist of target investors and get alerted when their interest signals align with a company you're monitoring.
</Note>

## API endpoints

Everything above is available through the Investors API. Full request and response detail for each endpoint lives in the [Investors API reference](/api/investors).

| Method | Endpoint | Returns |
|---|---|---|
| `GET` | `/investors` | Look up an investor by website or name |
| `GET` | `/investors/search` | Find an investor by name |
| `GET` | `/investors/{investorId}` | A full investor profile by ID |
| `GET` | `/investors/{investorId}/funding-rounds` | Funding rounds the investor has participated in |
| `POST` | `/investors/by-id/bulk` | Look up up to 50 investors in one request |

Investor interest signals are part of the Signals API — see [Investor interest signals](/api/signals).
