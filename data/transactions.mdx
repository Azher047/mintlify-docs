Specter's transactions database captures **1 million+ funding events and M&A transactions** across global private markets — from pre-seed cheques to billion-dollar buyouts. Each event is structured and consistent, so you can pull deal activity for a company, for an investor, or by a single round.

## What's Covered

  <Card title="Funding Rounds" icon="sack-dollar" href="/api/transactions">

  </Card>
  <Card title="M&A Events" icon="handshake" href="/api/transactions">

  </Card>
  <Card title="IPOs & Direct Listings" icon="arrow-trend-up" href="/api/transactions">

<Note>
  Where deal terms aren't publicly disclosed, amount and valuation fields come back `null`. The transaction itself — type, date, and investors — is still captured when there's enough public signal to confirm it happened.
</Note>

## API endpoints

Transactions are accessed by company, by investor, or by a transaction's own Specter ID. Full request and response detail for each endpoint lives in the [API reference](/api/transactions).

| Method | Endpoint | Returns |
|---|---|---|
| `GET` | `/companies/{companyId}/funding-rounds` | A company's funding rounds |
| `GET` | `/companies/{companyId}/acquisitions` | A company's acquisitions |
| `GET` | `/companies/{companyId}/ipos` | A company's IPOs |
| `GET` | `/investors/{investorId}/funding-rounds` | Funding rounds an investor participated in |
| `GET` | `/funding-rounds/{fundingRoundId}` | A single funding round by ID |
| `GET` | `/acquisitions/{acquisitionId}` | A single acquisition by ID |
| `GET` | `/ipos/{ipoId}` | A single IPO by ID |
