Specter's transactions database captures **1 million+ funding events and M&A transactions** across the global private markets — from pre-seed checks to billion-dollar leveraged buyouts. Every transaction is structured with consistent, queryable fields so you can analyze deal activity at the company level, across a portfolio, or market-wide. Whether you're tracking competitive funding velocity, benchmarking round sizes by sector, or sourcing acquisition targets, transactions data gives you the factual foundation for every deal-related conversation.

## Transaction Data Fields

Each transaction record in Specter includes a consistent set of structured fields regardless of transaction type:

| Field | Description |
|---|---|
| `id` | Unique Specter transaction identifier |
| `company_id` | The company that raised or was involved in the transaction |
| `type` | Round type: `pre_seed`, `seed`, `series_a`, `series_b`, `series_c`, `series_d`, `growth_equity`, `acquisition`, `ipo` |
| `amount_usd` | Total capital raised or deal value in USD |
| `date` | Transaction close or announcement date |
| `investors` | Array of participating investor IDs and names |
| `lead_investor` | Lead investor name and ID (where identifiable) |
| `post_money_valuation_usd` | Post-money valuation at close, where disclosed |
| `source` | Public disclosure, press release, or regulatory filing |

## API Reference

Query transactions with flexible filters to pull exactly the deal data you need.

```bash
GET https://app.tryspecter.com/api/v1/transactions
```

**Headers**

| Header | Value |
|---|---|
| `X-API-Key` | Your Specter API key |

**Query Parameters**

| Parameter | Type | Description |
|---|---|---|
| `company_id` | string | Filter to transactions involving a specific company |
| `type` | string | Filter by transaction type (e.g. `series_a`, `acquisition`) |
| `from_date` | date | Return transactions on or after this date (`YYYY-MM-DD`) |
| `to_date` | date | Return transactions on or before this date (`YYYY-MM-DD`) |
| `investor_id` | string | Filter to transactions where a specific investor participated |
| `min_amount_usd` | integer | Minimum transaction amount in USD |
| `max_amount_usd` | integer | Maximum transaction amount in USD |
| `limit` | integer | Number of results to return (default: `25`, max: `100`) |
| `offset` | integer | Pagination offset |

**Example Request**

```bash
curl -X GET "https://app.tryspecter.com/api/v1/transactions?company_id=comp_01hx9z3kp2&type=series_a&from_date=2024-01-01" \
  -H "X-API-Key: sk_live_yourApiKeyHere"
```
