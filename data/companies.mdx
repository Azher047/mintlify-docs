The Companies API gives you direct access to Specter's full company intelligence layer. Each profile aggregates structured data across funding rounds, hiring trends, web traffic, revenue estimates, social traction, and proprietary growth signals — all normalized and ready to use in your workflows.

## Get a Company Profile

**`GET /api/v1/companies/{companyId}`**

Retrieve the full enriched profile for a single company by its Specter ID.

### Path Parameters

<ParamField path="companyId" type="string" required>
  The unique Specter ID for the company. You can obtain this from a search result or a previously stored record.
</ParamField>

### Example Request

```bash
curl --request GET \
  --url "https://app.tryspecter.com/api/v1/companies/cmp_01h9xk2r4n5t8v3w" \
  --header "X-API-Key: YOUR_API_KEY"
```

### Response Fields

<ResponseField name="id" type="string">
  Specter's unique identifier for the company. Use this ID when querying related endpoints such as transactions or signals.
</ResponseField>

<ResponseField name="organization_name" type="string">
  The company's official trading or registered name.
</ResponseField>

<ResponseField name="description" type="string">
  A structured summary of the company's product, market, and value proposition.
</ResponseField>

<ResponseField name="industries" type="array">
  A list of industry and sector tags applied to the company, derived from Specter's classification taxonomy.
</ResponseField>

<ResponseField name="operating_status" type="string">
  Current operational status of the company. One of: `active`, `inactive`, or `acquired`.
</ResponseField>

<ResponseField name="employee_count" type="integer">
  The most recent headcount estimate based on aggregated signals including LinkedIn and job posting data.
</ResponseField>

<ResponseField name="growth_stage" type="string">
  Specter's proprietary funding and maturity stage classification. One of: `pre_launch`, `early_stage`, `growth_stage`, or `late_stage`.
</ResponseField>

<ResponseField name="highlights" type="array">
  A list of signal labels that flag notable activity or characteristics. Examples include `headcount_surge`, `top_tier_investors`, `revenue_signal`, and `recent_acquisition`.
</ResponseField>

<ResponseField name="funding" type="object">
  Aggregated funding data for the company.
  <Expandable title="funding fields">
    <ResponseField name="total_funding_usd" type="integer">
      Total capital raised across all rounds, in USD.
    </ResponseField>
    <ResponseField name="last_funding_date" type="string">
      ISO 8601 date of the most recent funding event.
    </ResponseField>
    <ResponseField name="last_funding_type" type="string">
      Round type of the most recent funding event, e.g. `series_b`, `seed`, `growth_equity`.
    </ResponseField>
    <ResponseField name="round_count" type="integer">
      Total number of tracked funding rounds.
    </ResponseField>
    <ResponseField name="post_money_valuation_usd" type="integer">
      Post-money valuation in USD from the most recent round where disclosed. `null` if undisclosed.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="hq" type="object">
  Headquarters location data.
  <Expandable title="hq fields">
    <ResponseField name="city" type="string">
      City where the company is headquartered.
    </ResponseField>
    <ResponseField name="state" type="string">
      State or province, where applicable.
    </ResponseField>
    <ResponseField name="country" type="string">
      Country of headquarters.
    </ResponseField>
    <ResponseField name="region" type="string">
      Broad geographic region, e.g. `North America`, `Europe`, `Southeast Asia`.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="revenue_estimate_usd" type="integer">
  Specter's modeled annual revenue estimate in USD, derived from web traffic, headcount, funding, and industry benchmarks.
</ResponseField>

<ResponseField name="socials" type="object">
  Social media presence and follower counts.
  <Expandable title="socials fields">
    <ResponseField name="twitter" type="object">
      Twitter/X handle and current follower count.
    </ResponseField>
    <ResponseField name="linkedin" type="object">
      LinkedIn company page URL and follower count.
    </ResponseField>
    <ResponseField name="instagram" type="object">
      Instagram handle and follower count.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="web" type="object">
  Website traffic and engagement metrics.
  <Expandable title="web fields">
    <ResponseField name="visits" type="integer">
      Estimated monthly unique website visits.
    </ResponseField>
    <ResponseField name="popularity_rank" type="integer">
      Global website popularity rank. Lower values indicate higher traffic.
    </ResponseField>
    <ResponseField name="bounce_rate" type="number">
      Ratio of single-page sessions to total visits, expressed as a decimal (e.g. `0.41` = 41%).
    </ResponseField>
    <ResponseField name="session_duration" type="integer">
      Average session duration in seconds.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="traction_metrics" type="object">
  Time-series growth metrics for key signals. Each metric object contains a `latest` absolute value and percentage change keys `1mo`, `3mo`, `6mo`, and `12mo`, expressed as decimals (e.g. `0.21` = +21%).
  <Expandable title="traction_metrics fields">
    <ResponseField name="web_visits" type="object">
      Monthly web visit trends, with `latest` count and change ratios for `1mo`, `3mo`, `6mo`, and `12mo` windows.
    </ResponseField>
    <ResponseField name="employee_count" type="object">
      Headcount trends, with `latest` count and change ratios for `1mo`, `3mo`, `6mo`, and `12mo` windows.
    </ResponseField>
    <ResponseField name="linkedin_followers" type="object">
      LinkedIn follower growth, with `latest` count and change ratios for `1mo`, `3mo`, `6mo`, and `12mo` windows.
    </ResponseField>
  </Expandable>
</ResponseField>

## Error Codes

| Status | Meaning |
|--------|---------|
| `200` | Success — company profile returned |
| `401` | Missing or invalid API key |
| `403` | Your plan does not include access to this endpoint |
| `404` | No company found for the provided ID |
| `429` | Rate limit exceeded — slow down your request cadence |
