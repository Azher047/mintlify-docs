Specter is an AI-powered private markets intelligence platform trusted by 300+ investment firms, with real-time data on 55M+ companies, 550M+ people, 330K+ investors, and 1M+ transactions. This page takes you from requesting access to your first live API call.

<Steps>
  <Step title="Request access">
    Specter is available to approved investment firms and operators. To get started, request access at [tryspecter.com/contact](https://www.tryspecter.com/contact). Once approved, you'll receive an invitation to set up your account on the Specter web app at [app.tryspecter.com](https://app.tryspecter.com).

    <Note>
      Not sure if Specter is right for your team? Reach out to [support@tryspecter.com](mailto:support@tryspecter.com) — the team is happy to walk you through a demo.
    </Note>
  </Step>

  <Step title="Get your API key">
    Once your account is active, open the **API Console** at [app.tryspecter.com/settings/api-console](https://app.tryspecter.com/settings/api-console). Copy your team's key — you'll use it to authenticate every request via the `X-API-Key` header. If there's no key yet, click **Request API Access** and it's approved the same day.

    <Tip>
      Keep your API key private and never commit it to source control. Use environment variables or a secrets manager to store it securely.
    </Tip>
  </Step>

  <Step title="Make your first call">
    The base URL for all requests is `https://app.tryspecter.com/api/v1`. Start by finding a company by name — this returns the basics, including the Specter ID you'll use to pull more.

    <CodeGroup>
      ```bash cURL
      curl --request GET \
        --url 'https://app.tryspecter.com/api/v1/companies/search?query=hedgehog' \
        --header 'X-API-Key: YOUR_API_KEY'
      ```

      ```python Python
      import requests

      response = requests.get(
          "https://app.tryspecter.com/api/v1/companies/search",
          params={"query": "hedgehog"},
          headers={"X-API-Key": "YOUR_API_KEY"}
      )
      results = response.json()
      print(results[0]["name"])
      ```

      ```javascript JavaScript
      const res = await fetch(
        'https://app.tryspecter.com/api/v1/companies/search?query=hedgehog',
        { headers: { 'X-API-Key': 'YOUR_API_KEY' } }
      );
      const results = await res.json();
      console.log(results[0].name);
      ```
    </CodeGroup>

    The response is an array of matching companies:

    ```json
    [
      {
        "id": "1a2b3c4d5e6g1a2b3c4d5e6g",
        "name": "Hedgehog Analytics",
        "domain": "hedgehog_analytics.com",
        "founded_year": 2019
      }
    ]
    ```

    If you get this back, your key works and you're talking to the API.
  </Step>

  <Step title="Pull the full profile">
    Take the `id` from that response and fetch the full company record — funding, headcount, location, and more.

    ```bash
    curl --request GET \
      --url https://app.tryspecter.com/api/v1/companies/1a2b3c4d5e6g1a2b3c4d5e6g \
      --header 'X-API-Key: YOUR_API_KEY'
    ```

    ```json
    {
      "id": "1a2b3c4d5e6g1a2b3c4d5e6g",
      "organization_name": "Hedgehog Analytics",
      "domain": "hedgehog_analytics.com",
      "hq": { "city": "London", "country": "United Kingdom" },
      "employee_count": 39,
      "funding": {
        "total_funding_usd": 16600000,
        "last_funding_type": "series_a"
      },
      "last_updated": "2026-01-01"
    }
    ```

    <Note>
      Search and enrich calls draw on your credit allowance. See the [API overview](/api/overview) for rate and credit limits.
    </Note>
  </Step>

  <Step title="Explore the platform">
    You're in. Specter offers multiple ways to surface private markets intelligence depending on your workflow:

    - **Web App** — Search, filter, and monitor companies, people, investors, and transactions at [app.tryspecter.com](https://app.tryspecter.com).
    - **REST API** — Build programmatic workflows and integrations using the full [API Reference](/api/overview).
    - **Chrome Extension** — Access Specter data inline as you browse LinkedIn, company websites, and more.
    - **CRM Integrations** — Sync Specter data directly into your CRM. See [CRM Integration](/integrations/crm).
    - **Specter MCP** — Use Specter's Model Context Protocol server to power AI agents with live private markets data.

    <Tip>
      Start with the [Company Data](/data/companies) and [Signals](/signals/overview) docs to understand the richness of data available across Specter's 55M+ company profiles.
    </Tip>
  </Step>
</Steps>
